# Machine Learning AG - Materialien

Willkommen im Repository für unsere Schüler-AG (Klassenstufen 9-11).

## Über dieses Projekt

Diese Sammlung umfasst interaktive Arbeits-Notebooks (Jupyter Notebooks), die speziell für den Einsatz in der AG konzipiert wurden. Ziel ist es, Schritt für Schritt in die Themen Programmierung, Datenanalyse und Künstliche Intelligenz einzuführen.

## Aufbau der Einheiten

Die Notebooks folgen einem klaren didaktischen Strukturprinzip:

1.  **Erklär-Teil**: Jeder Termin beginnt mit einem Abschnitt, der neue Konzepte anschaulich erklärt und Beispiele liefert.
2.  **Praxis-Teil**: Darauf folgt ein umfangreicher Aufgabenblock. Hier können die Schülerinnen und Schüler das Gelernte direkt anwenden, experimentieren und die Aufgaben eigenständig bearbeiten.

## Zeitrahmen

Jedes Notebook ist so dimensioniert, dass es ausreichend Material für einen **1,5-stündigen Termin** (90 Minuten) bietet.

## Inhalt

*   **01_jupyter-intro.ipynb**: Einführung in die Arbeitsumgebung und erste Schritte mit Python.
